# Clawd Splat

`Clawd Splat` is a small, single-file browser game prototype. Open
`index.html` in a browser and play a 90-second turf match with orange and teal
teams, procedural canvas rendering, keyboard/mouse controls, and mobile touch
controls.

## Play

- Desktop: 1P vs CPU or local 2P.
- Mobile: 1P vs CPU.
- Runtime: self-contained `index.html`, no external scripts, CDNs, images,
  audio files, or build step.
